Set the cmax parameter: lightwf

Calling Sequence
... 
  
lightwf (cmax)
Description
Sets the cmax parameter interactively, assuming the current 3D display list contains the result of a previous plwf call. This changes the color of the brightest surface in the picture. The darkest surface color can be controlled using the ambient keyword to light3 (see ``Lighting:'' on page ).
