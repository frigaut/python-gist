User Setup and Demo

At LLNL, the public version of python is installed in /usr/apps/python/opt/bin on both open and closed Livermore Computing computers and on the local AX-net computers. Ensure that python is in your path; if it is not, add: 

set path = ( /usr/apps/python/opt/bin $path )
to your .cshrc if your are a CSH user, and source the file.

There is a demonstration Python script installed in
${PYTHONHOME}/lib/python2.2/site-packages/gist/gistdemolow.py, where PYTHONHOME is where your Python is installed; at LLNL, it is /usr/apps/python/opt.

To run the demo, type: 

python
import gistdemolow
gistdemolow.run()
(ctrl-d)
