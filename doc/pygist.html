PYGIST


Python Interface to Gist Graphics



















Abstract

This document describes usage of the PYGIST and GIST3D Python modules. PYGIST enables users to access Yorick graphics functions from the Python scripting language. This module provides 2D plotting capability. The separate GIST3D modules provides a set of Python functions for interpreted 3D graphics.
